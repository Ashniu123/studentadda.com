Notes Organizer

Have you ever scanned lots of images from different textbooks for reference and then lost them?
Have you ever backed up your written notes in from of images and then lost them in your gallery?
Then Studentadda is the perfect place for you!
With our Notes Organizer you can easily convert your note's pictures into a pdf file or any other preferred file format for easy cross platform access.
You can organize these notes according to their subjects.

Events Planner
and
Organizer

Another student friendly feature provided by Studentadda is the Events Organizer.
This will take care of all your college events, keep track of your submission deadlines and display them on the dashboard.
With this you can easily store new calendar entries with features like setting default event durations depending on type of event. (Eg. If the type of event is practical then you can select this event type when creating an entry and then automatically 2hrs will be allotted from event start timeâ€¦ or if the type of event is lecture then default time allotted will be 1hr.)
Also you will be able to color code different appointments so it will be easy to view and distinguish between them.
For upcoming exams you will automatically be alerted about days left on the dashboard!

Dashboard

The Studentadda Dashboard will display both the Notes Organizer and Events Planner on a single interactive webpage.
The Dashboard will display all your uploaded notes in order you want or priority.
This priority will be based on your Events and Notes related to upcoming classes and exams will be shown first on the dashboard.
The Dashboard will also give you quick overview of all the upcoming events in a tabular form.
The entire month with important dates will be highlighted for easy reminders.
If a test is close then a reminder will be appear on the dashboard.
